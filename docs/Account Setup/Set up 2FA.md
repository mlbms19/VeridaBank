# How do I set up two-factor authentication (2FA)?

2FA adds a second layer of security to your account by requiring a one-time code in addition to your password. 

You can set up 2FA for additional verification through SMS, email, or an authenticator app.

**On VeridaBank Online**

1.	Go to **Settings** > **Security**.
2.	Select **Two-Factor Authentication (2FA)**.
3.  Toggle the **2FA switch** ![Toggle icon](toggle_on.png){ width="20" } to enable 2FA.
4.	Choose your verification method:<br>
    •&nbsp;&nbsp; **SMS**<br>
    •&nbsp;&nbsp; **Email**<br>
    •&nbsp;&nbsp; **Authenticator App**
5.	Enter the verification code received through your selected method.
6.	Click (or tap) **Confirm**.
